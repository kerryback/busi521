# BUSI 521 / ECON 505: Asset Pricing

## Overview
Course website for graduate asset pricing (Financial Economics I), Spring 2026 at Rice Business. Built with Quarto, deployed to GitHub Pages.

## Build
- `quarto render` renders the site to `docs/`
- `quarto preview` for local preview
- Deployed at busi521.kerryback.com

## Structure
- `*.qmd` files are pages (syllabus, slides index, assignments, exams)
- `Chapter*.pdf` are textbook chapter PDFs
- `latex/` contains LaTeX-generated PDFs
- `solutions/` contains assignment solution PDFs
- `docs/` is the rendered site (committed for GitHub Pages)

## Conventions
- Themes: flatly (light) / superhero (dark)
- External links open in new window
- Course meets TTh 2:30-3:45pm, McNair 104
- NotebookLM integration for student Q&A
